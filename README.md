# ETL_Project_Retail
Built an end-to-end Retail Sales ETL pipeline with Python, Pandas, and SQL Server. Extracted denormalized CSV data, transformed it into a relational model (Customer, Order, Purchase tables), and loaded it via SQLAlchemy. Enforced data integrity with DDL constraints and authored complex SQL queries to extract actionable business insights.

# Data folder
Data folder contains the three cleaned data csv files and one is raw data file.

# Python folder
This folder contains the Python commands to manupulate the raw data and perform data manupulation and droping the NULL values from it. Also done data normalization and store this raw data into three csv files with cleaned dataset. and then load the data to database through SQLALCHEMY.

# SQL folder
This folder contains the SQL queries to amke the columns primary and secondary keys and then perform multiple queries in order to show the insights of the data.
